# Workstation
Setup and dotfiles

## Manual steps

### iTerm2
* Preferences > Profiles > Colors > Color Presets > Import > [nord-iterm2](https://github.com/arcticicestudio/nord-iterm2) theme from `/iterm2_theme`
* Tweak ANSI colors
* Preferences > Profiles > Text > Font > size 14
